## Cross-Slide Observations

1. **Consistent template**: All 3 slides use the same 2-panel vintage curve layout
2. **Title bug**: Slides 2 and 3 still have "VCN Trigger and VDA Batch/Seasonal" in the subtitle — should be updated per slide
3. **Annotation format**: "{COHORT} Action {RATE}%" / "Lift {LIFT} bps -- {SS/Not SS}"
4. **Color coding in annotations**: Green = latest/best, Blue = middle cohorts, Red = oldest/worst (or most recent depending on context)
5. **30-day window** used for all charts (consistent with treatment window measurement)
6. **Campaign pairs by funnel position**:
   - Top: VCN (trigger, monthly) + VDA (batch, seasonal) — both acquisition
   - Middle: VDT (trigger, activation) + VUI (trigger, usage)
   - Bottom: VUT (email, provisioning) + VAW (in-app, provisioning)
